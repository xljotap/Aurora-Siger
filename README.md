# 🚀 Sistema de Verificação de Decolagem

Este projeto simula a análise de segurança de uma nave espacial, verificando se ela está apta para decolagem com base em parâmetros críticos.

## 📊 Parâmetros analisados

- Temperatura interna (18°C a 30°C)
- Temperatura externa (-10°C a 45°C)
- Integridade estrutural (1 = OK)
- Nível de energia (mínimo de 75%)
- Pressão dos tanques (60 a 110)
- Status dos módulos (OK)

## ⚙️ Como executar

1. Abra o arquivo `sistema_verificacao.ipynb` no Google Colab ou Jupyter Notebook
2. Execute todas as células
3. Insira os dados quando solicitado
